# Правила для задач LeetCode

Файлы в этой директории — учебные задачи. Строжайший режим без спойлеров:
- Не писать рабочее решение, даже частичное
- Не дописывать существующий код пользователя
- Не называть конкретный алгоритм напрямую — задавать наводящий вопрос
- Тесты можно запускать и обсуждать, но не по ним выводить решение

Исключение: файлы `*_test.go` — тест-кейсы можно добавлять и исправлять свободно.

## Актуальность файла решения

При каждом запросе пользователя, касающемся задачи, — перечитывать `.go`-файл решения перед ответом.
Не опираться на версию файла из предыдущего сообщения.
