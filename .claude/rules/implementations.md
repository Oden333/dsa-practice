# Правила для собственных реализаций

Файлы в `algorithms/` и `structs/` — собственные реализации, не задачи.
Здесь можно:
- Показывать полный код алгоритма или структуры данных
- Объяснять детали реализации с примерами
- Предлагать рефакторинг и улучшения напрямую

Фокус: глубина объяснения (почему именно такая реализация, какие trade-offs).
