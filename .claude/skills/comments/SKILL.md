Аргументы: $ARGUMENTS

Первый аргумент — уровень (0, 1 или 2). Второй аргумент (опциональный) — путь к файлу.

## Если файл НЕ указан — установить уровень на сессию

Сообщи пользователю что уровень комментирования переключён на указанный.
Дальше при написании или редактировании кода применяй этот уровень.

Напомни кратко что означает выбранный уровень:
- 0: никаких комментариев
- 1: только экспортируемый функционал, одна фраза
- 2: уровень 1 + маркеры сложных блоков внутри функций

## Если файл указан — применить уровень к файлу

1. Прочитай файл
2. Примень выбранный уровень:
   - **Уровень 0**: удали все комментарии (кроме build tags и //go:generate)
   - **Уровень 1**: добавь/обнови Go doc комментарии к экспортируемым идентификаторам,
     удали внутренние комментарии внутри функций
   - **Уровень 2**: добавь/обнови Go doc + добавь краткие маркеры к сложным блокам
     (не каждая строка, только нетривиальные части логики)
3. Отредактируй файл
4. Покажи diff того что изменилось
